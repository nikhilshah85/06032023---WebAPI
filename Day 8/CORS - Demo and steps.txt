CORS - Cross Origin Resource Shareing
	When 2 domains wants to exchange content / (data in our case) - there should be rights established otherwise 
			calls are blocked

	with .net core we can have CORS policy to define which website gets to access which contents

	with .net core we have to configure the service for the same and add a middleware
